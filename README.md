## GitHub Pages 배포

이 프로젝트는 정적 사이트(`index.html`, `styles.css`, `script.js`)로 변환되어 있습니다.

1. `index.html`, `styles.css`, `script.js`, 그리고 `images/` 폴더를 GitHub 저장소에 커밋하세요.
2. GitHub 저장소 설정(Settings)에서 Pages를 선택합니다.
3. 브랜치는 `main` 또는 현재 기본 브랜치를 선택하고, 폴더는 `root`를 선택합니다.
4. 저장 후 GitHub Pages URL에서 사이트를 확인합니다.

> `gh-pages` 브랜치를 사용하는 경우, 해당 브랜치에 정적 파일을 배포하면 됩니다.
